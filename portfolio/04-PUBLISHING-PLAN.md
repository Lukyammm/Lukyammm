The order, assets, captions, and CTAs to take this portfolio public. Nothing here
gets published without your approval — this is the playbook, you press the button.

## Phase 0 — GitHub first (do this before any LinkedIn post)

1. Merge this branch's profile README + portfolio package.
2. Paste each new README from `portfolio/readmes/` into its repo (replace the
   generic one). Commit message suggestion: `docs: rewrite README for recruiters`.
3. Capture screenshots per `03-SCREENSHOT-GUIDE.md`, commit them to each repo's
   `docs/assets/`.
4. Apply descriptions, topics, and pinned repos from `02-GITHUB-SETUP.md`.
5. Delete/hide the empty repos (`coordena_planilhas_gestao`, `INFECTOLOGIA`).

A LinkedIn post sends recruiters to GitHub — so GitHub has to be ready first.
